# goit-pycore-hw-04

Домашнє завдання. Тема 6. Робота з файлами та модульна система


## Запуск

Ініціалізуція віртуального середовища
```bash
source .venv/bin/activate
```

Запуск всього разом
```bash
python main.py
```

Запуск кожного завдання окремо
```bash
python task1.py
python task2.py
python task3.py <шлях_до_директорії>
python task4.py
```
